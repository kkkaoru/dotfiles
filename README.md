# Setup with Dotfiles

## Node.js

Use nodenv from anyenv

### Setup anyenv

```
anyenv install --init
```

## Install nodenv

```
anyenv install nodenv
```


## SHELL

Use fish with oh-my-fish


### Setup oh-my-fish

[Official install way](curl -L https://get.oh-my.fish | fish)

```
curl -L https://get.oh-my.fish | fish
```
