# Setup with Dotfiles

## Node.js

Use nodenv from anyenv

### Setup anyenv

```
anyenv install --init
```

## Install nodenv

```
anyenv install nodenv
```


## SHELL

Use fish with oh-my-fish


### Setup oh-my-fish

[How to install the official](https://github.com/oh-my-fish/oh-my-fish#installation)

```
curl -L https://get.oh-my.fish | fish
```
